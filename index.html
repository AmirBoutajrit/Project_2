HTML Basics

Every HTML document starts with a basic structure. Here's what you need:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Website</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first website.</p> </body> </html>

Common HTML Tags

<h1> to <h6> - Headings (h1 is largest)

<p> - Paragraphs

<a href="url"> - Links

<img src="image.jpg"> - Images

<div> - Container for grouping elements

<nav> - Navigation menu

<section> - Section of content

Pro Tip: Always close your HTML tags! What you open with <tag> must close with </tag>

CSS Styling

CSS makes your website look amazing! You can add CSS in three ways:

1. Inline CSS (not recommended)

<p style="color: blue;">This text is blue</p>

2. Internal CSS (in the <head> tag)

<style> p { color: blue; font-size: 18px; } </style>

3. External CSS (best practice)

Create a separate style.css file and link it:

<link rel="stylesheet" href="style.css">

CSS Selectors

/* Element selector */ h1 { color: purple; } /* Class selector */ .my-class { background-color: lightblue; } /* ID selector */ #my-id { font-size: 24px; }
Remember: Use classes (.) for reusable styles and IDs (#) for unique elements!
